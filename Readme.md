#### Steps after creating repository

- `npm init -y`
- `npm install cypress`
- `npm install`

- add Readme.md
- add `.gitingnore`
- add to gitignore: .idea, node-modules
- git commit, git push

- `npx cypress open` will create structure of cypress folders (tests -> integration folder)

#### Run the tests:

- `npm run start` or `npm cypress open`
